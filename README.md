# Modern Unit Converter

A sleek and user-friendly Android app built with **Jetpack Compose** to convert units of length, including meters, centimeters, feet, and millimeters.

## Features
- Convert values between **Meters, Centimeters, Feet, and Millimeters**
- Modern and intuitive UI with **Material3 design**
- Real-time conversion with **precise results up to 4 decimal places**
- Simple input and output selection via dropdowns

## Technology Stack
- **Kotlin** – programming language
- **Android Studio** – IDE for development
- **Jetpack Compose** – for modern, declarative UI
- **Material3** – for styling and theming
